# Expense Tracker Web App

## 📌 Overview

This project is a simple web-based **Expense Tracker** application that allows users to manage their personal expenses. Users can add, edit, delete, and view expenses directly in the browser.

The goal of this project is to demonstrate understanding of web application development using **HTML, CSS, and JavaScript**, including DOM manipulation and local data storage.

---

## 🚀 Features

* Add new expenses
* Edit existing expenses
* Delete expenses
* View all expenses in a table
* Automatic total calculation
* Persistent data using LocalStorage

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage (browser storage)

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the `index.html` file in your browser
3. Start adding and managing expenses

No installation or server is required.
